## Define a Kill Date for Every Feature Flag

- **Context**: Każda faza dodająca feature flag
- **Problem**: Flagi bez daty usunięcia pozostają na zawsze i zamieniają się w dług techniczny.
- **Rule**: Always define a kill date when creating a feature flag.
- **Applies to**: plan, implement, impl-review
